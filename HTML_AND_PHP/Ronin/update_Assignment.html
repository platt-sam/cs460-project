Assignment Updates

Author:Ronin Ganoot

This page provides an example interface to show that the after_update_assignment trigger is functioning correctly.

Description:

Checks if new Assignments have been inserted into the table to keep a log of homework on a table that stores it.



Justification:

Will help students tell which homework has been assigned and keep track of that.
