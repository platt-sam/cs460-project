Assignment Tracker

[View] Tasks to a Student

Author: Nayeli Esqueda Alvarado

Description:

This view will display all assignments found under the task table for a student when a student id is inputted.



Justification:

This is useful for students, in that the end result will display all their assignments that can be found under their student id.


Please enter a student id in order to view all tasks. (Ex. Student ID = 94039013)
